- Create a new repo, push the folder, enable GitHub Pages from settings (branch: main, folder: root/gh-pages).


Option B: Vercel
- Connect a GitHub repo and deploy (auto-detect static). Works with zero config.


Option C: Netlify
- Drag & drop site folder into Netlify dashboard.


## 6) Accept payments
Create a Gumroad/LemonSqueezy listing, upload the zipped folder as the product, and set price.
